# 🐍 Curso Python

## 📖 Sobre o Repositório

Este repositório foi criado para documentar minha jornada de aprendizado em Python, reunindo exercícios, exemplos práticos, anotações e projetos desenvolvidos ao longo dos estudos.

O objetivo é construir uma base sólida em programação, análise de dados e automação, registrando minha evolução desde os conceitos fundamentais da linguagem até aplicações mais avançadas.

* Aprender os fundamentos da linguagem Python.

* Praticar resolução de problemas e algoritmos.

## 🚀 Como Executar os Arquivos

1. Clone este repositório:

```bash
git clone git@github.com:charlitonSN/curso-python.git
```

2. Acesse a pasta do projeto:

```bash
cd curso-python
```

3. Execute qualquer arquivo Python:

```bash
python nome_do_arquivo.py
```
